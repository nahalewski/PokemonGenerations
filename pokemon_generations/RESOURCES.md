# Project Resources

This document maintains a list of key external resources and repositories utilized for data, maps, and world-building in the Pokémon Generations project.

## Core Data Sources
- **[poketrax/pokedata](https://github.com/poketrax/pokedata)**: Comprehensive Pokémon datasets including base stats, growth rates, and move learning tables.
- **[PokeAPI](https://pokeapi.co/)**: RESTful API utilized for dynamic Pokémon data fetching.
- **[OpenWeatherMap](https://openweathermap.org/)**: External API providing real-time local weather data (Rain, Snow, Cloud cover) for environmental synchronization.
- **[speech_to_text](https://pub.dev/packages/speech_to_text)**: Flutter plugin enabling natural language command recognition in the Battle HUD.

## Mapping & World Building
- **[pkmnmap.com/FireRedLeafGreen](https://pkmnmap.com/FireRedLeafGreen/)**: Interactive maps for the Kanto region, used as primary reference for the Story Mode/World Map development.

## Assets & Themes
- **Pokémon Home Aesthetic**: Primary visual inspiration for the Sidebar and UI overhaul (v2.1).
- **HGSS TileSets**: Reference for starting town and exterior mapping logic.
